client->server packet
| idx | description
|-----|------------
|  0  | byte -- Packet command
| 1-16| byte[] -- current packet

| idx | description
|-----|------------
|  0  | byte -- Refresh time command
| 1-4 | int -- new refresh time


Commands
| byte | description
|------|------------
| 0x01 | Packet
| 0x02 | Frame complete
| 0x03 | Set refresh time
